# 四、OBS Studio 30.2.3 常用快捷键（提高操作效率，新手必记）



1.**开始 / 停止录制**：Ctrl + Shift + R（录屏的时候不用鼠标点 “开始录制” 按钮，按这个快捷键特别方便，尤其是玩游戏的时候，不用切出来点鼠标）；

2.**暂停录制**：Ctrl + Shift + P（录网课、开线上会议时，中间需要暂停一下，按这个快捷键就行，不用结束录制，后续还能接着录）；

3.**开始 / 停止直播**：Ctrl + Shift + S（直播的时候想快速开始或停止推流，按这个快捷键比点界面上的按钮快多了）；

4.**切换场景**：Ctrl + 数字键（比如你创建了 “场景 1”“场景 2”，按 Ctrl+1 就能快速切换到场景 1，直播换画面的时候很实用）；

5.**打开设置界面**：Ctrl + ,（逗号键，想调录制清晰度、帧率这些参数，按这个快捷键能直接打开设置，不用在界面上找 “设置” 按钮）；

6.**添加新来源**：Ctrl + Shift + N（要加摄像头、捕获屏幕画面，按这个快捷键能快速打开添加来源的窗口，比右键点 “来源” 面板方便）。

记住这些常用快捷键，平时用 OBS 录屏或直播的时候，操作会更顺手，效率也能提高不少～
